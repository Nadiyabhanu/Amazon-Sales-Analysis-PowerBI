# 📊 Amazon Sales Analysis Dashboard

## 📌 Overview
This project analyzes Amazon sales data (2019–2020) using Power BI to identify trends, regional performance, and profitability.

## 🎯 Objectives
- Analyze sales and profit performance
- Identify top-performing regions and categories
- Understand sales trends over time

## 📈 Key Insights
- West region generates highest profit
- Technology category leads in sales
- Furniture category has low profit margins
- Sales show fluctuating trend

## 💡 Business Recommendations
- Focus on high-profit regions
- Improve pricing in low-margin categories
- Optimize discount strategies

## 🛠 Tools Used
- Power BI
- Excel
